# ES6-demo
### 使用
#### a. 命令行执行<br>
     npm start
#### b. 浏览器上输入 http://localhost:9000
#### c. 去 index.js 去编写代码吧，浏览器会自动刷新，不要忘了打开Chrome浏览器调试面板看输出。（按F12即可）
